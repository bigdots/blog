java控制流程语句

Java 主要的控制语句有3 种，选择语句、循环语句、跳转语句。

条件语句

  • if(){}else{}
  • if(){}else if(){}else{}
  • switch

循环语句

  • while

  • do...while

  • for

    跳转语句

  • break

  • continue

  • return
