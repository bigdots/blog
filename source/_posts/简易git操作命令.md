# git常用操作命令

### 新建本地仓库（俩种方式）
1. 本地新建文件夹，然后初始化

 ``` 
 git init
 ```
 
2. 直接创建并初始化文件夹

 ```
   git init <reponName> 
 ```  

### 添加新文件
```
git add <filename>
```

### 更新本地代码

```
git pull
```

### 比较代码

```
git diff <source> <target>
```

### 合并代码
```
git merge <target>
```

### 提交版本
```
git commit -m " commit info "
```

### 推送到远程仓库
```
git push origin master
```
